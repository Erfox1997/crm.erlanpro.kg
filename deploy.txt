# Если composer ругается на vendor/* — переустановите зависимости с нуля:

npm ci

#    Verify Token: значение META_WEBHOOK_VERIFY_TOKEN из .env
#    Подписка: messages (instagram)
# 3. Facebook Login → Settings → Valid OAuth Redirect URIs:
#    https://crm.erlanpro.kg/integrations/instagram/oauth/callback
#    https://crm.erlanpro.kg/integrations/facebook/oauth/callback
# 4. Настройки приложения → Основное:
#    - Добавить платформу «Веб-сайт», Site URL: https://crm.erlanpro.kg
#    - Домены приложений: crm.erlanpro.kg
#    - Заполнить URL политики конфиденциальности (обязательно для OAuth)
# 5. Роли приложения → добавить ваш Facebook-аккаунт как Администратор/Разработчик
#    (Instagram Tester недостаточно для Facebook Login)
# 6. .env на сервере — App ID из «Основное» (должен совпадать с CRM → Интеграции):
#    APP_URL=https://crm.erlanpro.kg
#    INSTAGRAM_APP_ID=...
#    INSTAGRAM_APP_SECRET=...
#    META_OAUTH_PROVIDER=facebook
#    META_WEBHOOK_VERIFY_TOKEN=crm-ulan-meta-webhook
# 6. В CRM: Интеграции → Отключить → Подключить через Meta
#    (OAuth через Facebook, токен страницы + Page API)
