Техническое задание (ТЗ) Проект: crm.ulan.kg

1.  Общая информация Тип: Web CRM система (SaaS, multi-tenant) Backend:
    Laravel База данных: MySQL Frontend: Blade / Vue (рекомендуется Vue
    3 + Vite) API: REST

Данные базы: ulancrm, ulancrm, QYT07RR2KaK9


2.  Роли пользователей

2.1 Администратор - Управление компаниями - Управление тарифами -
Просмотр статистики - Управление пользователями - Настройка интеграций

2.2 Пользователь - Регистрация / авторизация - Личный кабинет -
Управление своей компанией - Создание менеджеров - Работа со сделками и
клиентами

3.  Архитектура Одна база данных Одна система Изоляция через company_id
    во всех таблицах

4.  Основные модули

4.1 Регистрация и авторизация - Регистрация через email - Подтверждение
email - Восстановление пароля

4.2 Личный кабинет - Дашборд (сделки, доход, клиенты)

4.3 Управление компанией - Название - Тариф - Настройки - Сотрудники

4.4 Менеджеры - Добавление / удаление - Роли - Ограничения доступа

4.5 Воронки продаж - Создание воронок - Этапы - Перетаскивание сделок

4.6 Сделки - Создание сделки - Клиент - Сумма - Статус - Ответственный

4.7 Клиенты - ФИО / название - Телефон - Email - История

4.8 Тарифы - Бесплатный - Стандарт - Премиум

4.9 Навигация - Левое меню

4.10 Интеграции - WhatsApp - Instagram - TikTok (будущее) - Lalafo
(будущее)
